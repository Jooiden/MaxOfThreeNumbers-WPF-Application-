# MaxOfThreeNumbers (WPF Application)

Приложение с графическим интерфейсом (WPF) на платформе .NET Framework 4.7.2, предназначенное для автоматического нахождения максимального значения среди трёх введённых целых чисел.

## 🚀 Функционал
* **Валидация ввода:** Программа проверяет, все ли поля заполнены, и выводит предупреждение, если пользователь оставил поля пустыми.
* **Обработка ошибок:** Защита от ввода некорректных данных (буквы, спецсимволы, дробные числа) с помощью блока `try-catch` и вывода информативного окна ошибки.
* **Управление интерфейсом:** Возможность очистки только полей ввода (`Очистить`) или полного сброса приложения вместе с полученным результатом (`Сброс`).

## 🛠 Технологический стек
* **Язык программирования:** C#
* **Технология интерфейса:** WPF (Windows Presentation Foundation) / XAML
* **Платформа:** .NET Framework 4.7.2

## 📁 Структура проекта
* `MainWindow.xaml` — разметка графического интерфейса пользователя (поля ввода, кнопки, текстовые блоки).
* `MainWindow.xaml.cs` — логика обработки событий (нажатия кнопок, валидация, парсинг и расчёт).
* `WpfApp1.csproj` — файл конфигурации проекта MSBuild.
